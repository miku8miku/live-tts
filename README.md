# Live TTS

一个基于 Google Gemini 模型的实时文本转语音应用，融合了 AI 驱动的语音交互与趣味游戏体验。

本项目是阿里云 ESA (Edge Security Acceleration) 边缘计算创新大赛的创意参赛作品。借助 阿里云 ESA强大的边缘计算能力，我们将这个富交互的静态网站分发到全球节点，确保用户在世界任何角落都能体验到闪电般的加载速度。


## 项目亮点

### 创意卓越
该项目巧妙地将实时文本转语音（TTS）功能与互动游戏体验相结合，创造了一个独特的 AI 驱动的"猜词游戏"体验。项目不仅提供了基础的 TTS 功能，还创新性地开发了一个基于 Gemini 模型的智能游戏系统，让用户可以通过语音与 AI 进行猜词互动。界面设计采用现代化的暗黑模式，并包含动态音频电平指示器，提供了视觉与听觉结合的丰富交互体验。

### 应用价值
1. **多语言支持**：项目支持包括中文、英文、日文、韩文在内的超过 25 种语言，满足国际化需求
2. **实用工具价值**：提供高质量的文本转语音服务，可用于阅读辅助、内容创作、语言学习等多种场景
3. **教育娱乐价值**：内置的猜词游戏功能可作为语言学习工具，帮助用户在互动中提升语言能力
4. **灵活配置**：用户可选择多种预设声音（如 Zephyr、Achernar 等星体命名的声音），并自定义语音参数

### 技术探索
1. **前沿 AI 集成**：利用 Google 的 Gemini 2.5 Flash Native Audio 模型实现高质量实时语音合成和识别
2. **Web Audio API 深度运用**：实现音频流处理、实时音频分析、动态音量检测等高级功能
3. **实时流式通信**：构建了完整的 WebSocket 实时通信架构，支持低延迟的语音数据传输
4. **双模式会话管理**：分别实现了 TTS 专用会话和游戏双向交互会话，针对不同场景优化
5. **现代前端技术栈**：基于 React 19、TypeScript、Vite 构建，采用组件化架构设计

## 功能特性

- **实时文本转语音**：将输入的文本即时转换为自然流畅的语音输出
- **多语言支持**：涵盖全球主流语言，满足国际化的使用需求
- **多样化声音选择**：提供多种预设声音，包括以星体命名的独特声音选项
- **智能猜词游戏**：与 AI 进行语音互动的猜词游戏，增加娱乐性和互动性
- **动态音频可视化**：实时显示音频电平，增强用户体验
- **响应式设计**：适配各种设备尺寸，提供一致的用户体验
- **主题切换**：支持明暗两种主题模式，适应不同环境下的使用

## 快速开始

### 环境准备
确保您的开发环境中安装了以下软件：
- Node.js (版本 >= 18.0.0)
- npm 或 yarn 包管理器

### 安装步骤
1. 克隆项目到本地：
   ```bash
   cd gemini-live-tts
   ```

2. 安装依赖包：
   ```bash
   npm install
   # 或者使用 yarn
   yarn install
   ```

3. 设置环境变量：
   在项目根目录创建 `.env` 文件，并添加您的 Google Gemini API 密钥：
   ```
   API_KEY=your_google_gemini_api_key_here
   ```

4. 启动开发服务器：
   ```bash
   npm run dev
   # 或者使用 yarn
   yarn dev
   ```

5. 打开浏览器访问 `http://localhost:5173` 查看应用

### 生产构建
如需构建生产版本：
```bash
npm run build
# 或者使用 yarn
yarn build
```

## 解读规则

本项目基于 Google Gemini 的实时音频 API，实现文本到语音的转换以及语音游戏互动功能。以下是关键的技术实现规则：

### 文本转语音规则
- 输入文本会被完整地传递给 Gemini 模型进行语音合成
- 系统会按顺序处理音频块，确保语音输出的连续性
- 支持长文本的分段处理，避免模型限制

### 游戏互动规则
- 在猜词游戏中，AI 扮演描述者的角色，用户需要根据提示猜测目标词汇
- 系统会实时监听用户语音输入，并将其转换为文本进行处理
- 用户不能说出目标词本身或禁忌词，否则游戏结束
- 每轮游戏限时60秒，时间结束时若未猜出则游戏结束

### 音频处理规则
- 音频采样率为 24kHz，确保高质量的语音输出
- 实时计算音频电平，用于界面可视化反馈
- 音频播放采用 Web Audio API，确保低延迟体验
